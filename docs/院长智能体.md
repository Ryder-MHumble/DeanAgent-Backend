# 院长智能体

简单写了一些需求，不见得全面，也不见得都需要实现，供大家讨论：

*   对国家
    
    *   国家发布的教育、科技、人才、产业、成果转化等相关的最新政策，有哪些需要关注
        
    *   相关的国家领导人、部委领导的人事变动，哪些和学院发展相关
        
    *   发改委、教育部、科技部领导的相关讲话，有哪些需要关注
        
*   对北京
    
    *   北京发布的教育、科技、人才、产业、区域发展、成果转化相关的最新政策，有哪些值得关注
        
    *   相关的市领导、委办局的人事变动，哪些与学院发展相关
        
    *   市政府、相关委办局领导的相关讲话，有哪些值得关注
        
*   对技术
    
    *   国内外最新技术进展和趋势预测，其对于AI技术路线的新启示、对相关产业的新影响 -外网的热点技术话题和事件、KOL的最新发言
        
    *   预测领导会关心的技术问题，内参的机会
        
*   对人才
    
    *   哪些AI人才近期有回国意向，值得关注
        
    *   有哪些人才相关的指数、排名等发生新的变化，值得关注
        
*   对产业
    
    *   重大产业新闻及其启示
        
    *   AI相关创业企业及其融资、IPO相关信息
        
    *   预测领导会关心的产业问题，内参的机会
        
*   对学院
    
    *   学院内的项目进展，重大成果进展、提示机会和风险
        
    *   学院内部需要关注的人和事（学生情况、学生取得的最新成绩、学生思想和心理问题、上课情况、招生情况等等）
        
    *   小红书和社交媒体上与学院相关的话题，机会与舆情风险
        
    *   学院内部的key number的变化、key event的提醒，以及各类风险提示
        
    *   各中心进展、督办情况、风险提示
        
*   对高校
    
    *   相关高校的人事变动，需要发消息恭喜的人
        
    *   高校最新科技成果（获奖）、最新人才变动（院士、大人才，各种fellow），需要发消息恭喜的人
        
    *   兄弟单位动向（尤其是其他国智院，AI lab和清北），以及对学院发展的启示
        
*   对日程
    
    *   接下来几天要出席的重大活动，给出建议的材料（如发言稿）
        
    *   判断日程邀请的重要程度（组织的单位、同框嘉宾、活动的历史影响力），建议转给相关院领导参与
        
    *   建议哪些外部活动需要争取参加（近期技术诀窍、产业界的大型活动、媒体报告机会）
        
    *   发现日程冲突，建议调整方案
