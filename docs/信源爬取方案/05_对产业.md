## 5.0 产业信息传播链条分析

| 信息类型 | 传播链 | 最原始信源 | 当前覆盖 |
|----------|--------|-----------|---------|
| 国内融资（早期-B轮） | 企业/投资方PR → **36氪/投中网**首发 → 其他媒体跟进 | 36氪/投中网 | ✓ |
| 国内融资（间接信号） | 工商注册变更（增资/股权变更）→ 企查查/天眼查发现 → 媒体报道 | **国家企业信用信息公示系统** | ✓ |
| A股IPO | 企业递交招股书 → **交易所官网**受理公告 → 财经媒体报道 | **上交所/深交所/北交所** ✗缺失 | 公告当天 |
| 港股IPO（含AI中概股） | 企业递交招股书 → **港交所披露易**发布 → 财新/第一财经 → 全网 | **HKEX News** ✗缺失 | 公告当天 |
| 美股IPO（中概股） | 企业递交F-1/S-1 → **SEC EDGAR**公开 → 英文媒体 → 中文翻译 | **SEC EDGAR** ✗缺失 | 公告当天 |
| 国际融资 | 企业PR → TechCrunch/Crunchbase → 中文媒体翻译 | Crunchbase | ✓ |

> **结论**：融资新闻覆盖较好，但IPO信息存在显著盲区——A股/港股/美股IPO的首发源都是监管机构/交易所的官方公告，比媒体报道早数小时甚至更多。对于AI公司IPO这种高价值信息，建议直接监控交易所官方公告。

**免费方案说明**：
- 天眼查/IT桔子/企查查的直接爬取几乎不可行（极强反爬+法律风险），其付费API也暂不考虑
- **替代方案**：36氪融资频道 + 投中网免费板块可覆盖绝大部分AI融资新闻；Crunchbase免费版可覆盖国际部分
- 国家企业信用信息公示系统是免费官方数据源，可查工商变更（增资、股权变更等间接融资信号）
- **交易所/SEC方案说明**：三大A股交易所的信息披露页面均为免费公开数据，HTML结构简单。SEC EDGAR提供免费全文搜索API（无需认证），非常适合自动化。港交所披露易同样免费公开。这三个源可以覆盖AI企业IPO的全球首发信息。
