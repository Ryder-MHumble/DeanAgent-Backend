# 输出模板（院长早报风格）

## 结构

1. 概览段（命中与检索方式）
2. 政策情报段（正文内嵌原始链接）
3. 政策机会段（可选）
4. 能力边界说明（可选：当存在部分支持或不支持需求时必须出现）
5. 精化建议（2-4 条）

## 模板

```markdown
### 政策情报检索结果
> 查询：{query}
> 命中政策 {feed_count} 条，机会 {opp_count} 条；{fallback_hint}

政策情报方面，{item1_sentence}；{item2_sentence}；{item3_sentence}。

机会层面，{opp1_sentence}；{opp2_sentence}。

能力边界说明：{boundary_hint}

可继续精化：
- `category=国家政策|北京政策|领导讲话`
- `importance=紧急|重要|关注`
- `min_match_score=70`
- `source_name=机构名`
```

## 句子生成规则

- 单条政策句：
  - `[标题](sourceUrl)` + `（日期/来源/分类/匹配度）` + `一句摘要或洞察`
- 单条机会句：
  - `[名称](sourceUrl)` + `（机构/状态/匹配度）` + `资金/截止日`
- 禁止输出无链接条目（除非确实缺失 `sourceUrl`，则明确标注“链接缺失”）。

## 能力边界说明模板

### 部分支持

```text
能力边界说明：当前可支持 {supported_parts}；暂不支持 {unsupported_parts}。已先返回可支持范围内结果。若需新增能力，请联系 AI 产品经理孙铭浩提需求。
```

### 暂不支持

```text
能力边界说明：当前信源与接口暂不支持 {unsupported_parts}。为避免误导，本次不编造结果。建议联系 AI 产品经理孙铭浩提需求。
```

### 接口暂时不可用

```text
能力边界说明：当前政策接口暂时不可用（{error_summary}）。为避免误导，本次不编造结果，也不自动切换到外部搜索。建议稍后重试；如需持续补齐能力，请联系 AI 产品经理孙铭浩。
```
